This section contains the packages I use (and some I probably don't).

** Use Package
#+begin_src emacs-lisp
(unless (package-installed-p 'use-package)
			     (package-install 'use-package))

;; set up use-package
(require 'use-package)
(setq use-package-always-ensure t)

;; set up org-mode
(use-package org
	     :ensure t)

;; set up evil mode
(use-package evil
	     :ensure t
	     :config
	     (evil-mode 1))

#+end_src
